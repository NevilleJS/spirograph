# Spirograph
**Spirograph made in c++ using raylib** 


![](https://i.imgur.com/lu8rkUr.png)

**Usage**: 
```
git clone https://github.com/NevilleJS/spirograph.git
cd spirograph/build
cmake ..
cmake --build .
./spirograph
```
**NOTE:** Make sure you have installed raylib(https://github.com/raysan5/raylib)
